# Calculator

A Calculator for general purposes.

## Features

- Add
- Multiply
- Subtract
- Divide
- Square
- Square Root
- Power of n

## Built With

- HTML
- CSS
- JAVASCRIPT

## How to Run?

To run the application you need to clone the project and run the HTML file.
