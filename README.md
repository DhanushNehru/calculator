
# Calculator

A Calculator that performs many **mathematical** functions

![Mar-Business_11](https://user-images.githubusercontent.com/114678694/193993341-4915b37e-8de2-42a3-a940-8eb2b58aaedf.jpg)


## Features

```
- Add
- Multiply
- Subtract
- Divide
- Square
- Square Root
- Power of n
- Natural exponential function
- Factorial of n
```
